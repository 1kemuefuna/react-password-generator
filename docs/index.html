Reactify Password Generator

Description

Reactify Password Generator is a React component library for generating strong passwords and checking their strength.

Installation

You can install the library via npm or yarn. 

npm install reactify-password-generator

Usage

To use the Reactify Password Generator component, include it in your React application.

Example

Password Strength: 

  import react from 'react';
  import { reactifypasswordgenerator } from 'reactify-password-generator';

  const app = () => {
    const handlegeneratepassword = (password) => {
      console.log('generated password:', password);
    };

    return (
      <div>
        <h1>password generator</h1>
        
        <reactifypasswordgenerator
          rendertrigger={(props) => <button {...props}>generate password</button>}
          ongenerate={handlegeneratepassword}
          regex=/your-custom-regex-here/
        />

      </div>
    );
  };

  export default app;
      

  import react from 'react';
  import { reactifypasswordstrengthindicator } from 'reactify-password-generator';

  const app = () => {
    const password = 'your-password-here';

    return (
      <div>
        <h1>password strength indicator</h1>
        <reactifypasswordstrengthindicator password={password} regex=/your-custom-regex-here/ />
      /</div>
    );
  };

  export default app;

Props

You can configure the Reactify Password Generator component with various props, including:

  • renderTrigger: A function to render the password generation trigger button.
  • onGenerate: A callback function to handle the generated password.
  • regex: A regular expression to enforce password complexity (default: a combination of symbols, capital letters, small letters, and numbers).
